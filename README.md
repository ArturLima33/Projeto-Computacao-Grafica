# 🤖 Robô Interativo 2D

# 📝 Sobre o Projeto
Este projeto é uma cena 2D interativa criada com HTML, Canvas 2D e JavaScript. Ele apresenta um robô que pode ser controlado pelo usuário e um ímã que reage à presença do robô, alternando entre perseguição e fuga.

# 🛠️ Transformações Geométricas Utilizadas
O projeto aplica os três conceitos principais de transformações aprendidos em aula:

# Translação (translate):

Usada para mover o robô pelo cenário com as teclas WASD.

Usada para montar o corpo do robô, posicionando braços, pernas e cabeça em seus devidos lugares.

Cria o efeito de "câmera", movendo o fundo para que o robô fique no centro.

# Rotação (rotate):

- Braços e Cabeça: Seguem o movimento do mouse em tempo real.

- Caminhada: As pernas giram para frente e para trás conforme o robô se move.

- Ímã: O objeto fica girando constantemente no próprio eixo.

# Escala (scale):

- Animação: O corpo do robô "achata" e "estica" levemente ao andar para parecer mais vivo.

- Sombra: A sombra diminui de tamanho quando o robô pula, indicando altura.

# 🧲 Lógica do Ímã (Atração e Repulsão)
O ímã possui um comportamento automático baseado na distância do robô:

- Longe do Robô: O ímã tenta chegar perto do robô (Atração).

- Perto do Robô: Se o robô chegar muito perto, o ímã "se assusta" e foge para longe (Repulsão).

# 🎮 Como Controlar
- W, A, S, D: Move o robô para os lados, para frente e para trás.

- Espaço: Faz o robô pular.

- Mouse: Move a cabeça e os braços do robô.

# 🚀 Detalhes Adicionais
- Cenário: Prédios com luzes que piscam e chaminés que soltam fumaça.

- Física: O pulo tem gravidade, fazendo o robô cair de volta ao chão naturalmente.
